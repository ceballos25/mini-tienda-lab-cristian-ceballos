# 🛒 Mini Tienda con Carrito de Compras

Proyecto desarrollado como laboratorio de manipulación del DOM utilizando JavaScript, Bootstrap 5 y HTML.

Permite agregar productos al carrito, eliminarlos individualmente, calcular el total en tiempo real y vaciar completamente el carrito sin recargar la página.

---

## 🚀 Demo

🔗 https://ceballos25.github.io/mini-tienda-papitas/

---

## 🧰 Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5
* Bootstrap Icons
* SweetAlert2

---

## ⚙️ Funcionalidades

* 🛍️ Agregar productos al carrito
* ❌ Eliminar productos individualmente
* 🔢 Contador dinámico en el navbar
* 💰 Total actualizado en tiempo real
* 🧹 Vaciar carrito completo con confirmación
* 📱 Diseño responsive con Bootstrap
* 🔔 Notificaciones con SweetAlert2
